## ūüöÄ Overview

**CollabSphere** is a premium, high-performance mobile application built with **React Native** and **Expo**. It is designed for developers, designers, and builders to find squads and ship projects faster. 

The application features a modern, startup-grade UI with **60fps animations**, **Glassmorphism** components, and a custom **OLED-optimized dark theme**.
