## 🚀 Overview

**CollabSphere** is a premium, high-performance mobile application built with **React Native** and **Expo**. It is designed for developers, designers, and builders to find squads and ship projects faster. 

The application features a modern, startup-grade UI with **60fps animations**, **Glassmorphism** components, and a custom **OLED-optimized dark theme**.

## ✨ Key Features

- 💎 **Premium UI/UX**: Etheric Monochrome aesthetic with pure black backgrounds and sophisticated terracotta/sage/ochre accents.
- 🎬 **Motion System**: Integrated `react-native-reanimated` for fluid transitions, staggered entries, and interactive micro-animations.
- 💬 **Sticker-Style Chat**: A custom messaging interface with a unique sticker aesthetic, overlapping avatars, and geometric grid backgrounds.
- 🏗️ **Workstation Dashboard**: High-impact metrics bar and "sneaky peeker" user assets for a professional developer environment.
- 🕵️ **Social Feed**: Real-time collaborative feed with builder stories, shipping snippets, and technical interaction buttons.
- 🧭 **Discovery & Ranking**: AI-driven builder grids with matching percentages to find the ideal technical teammates.

## 🛠️ Tech Stack

- **Core**: React Native + Expo SDK 54
- **Language**: TypeScript
- **Animation**: React Native Reanimated (Spring-based)
- **Icons**: Lucide React Native
- **Styling**: Pure React Native StyleSheet + Expo Blur & Linear Gradient
- **Navigation**: React Navigation (Custom Tab Bar with sliding active bubble)

## 📦 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/Keerthanreddy01/Collabsphere-app.git
   cd Collabsphere-app
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npx expo start
   ```

4. **Run on Emulator**
   Press `a` for Android or `i` for iOS to launch the application.
